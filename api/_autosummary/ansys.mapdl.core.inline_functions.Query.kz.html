kz#

Query.kz(k)#

Return the z coordinate of a keypont.

Z-coordinate of keypoint k in the active coordinate system.

Parameters:
kint

Keypoint number to be considered.

Returns:
float

Coordinate of the keypoint.

Examples

Here we add a single keypoint, then use kz to extract the z-coordinate of it.

>>> from ansys.mapdl.core import launch_mapdl
>>> mapdl = launch_mapdl()
>>> mapdl.prep7()
>>> mapdl.k(1, 0, 1, 2)
>>> mapdl.queries.kz(1)
2.0
Return type:

float

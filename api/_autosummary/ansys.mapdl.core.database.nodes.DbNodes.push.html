push¶

DbNodes.push(inod, x, y, z, xang=None, yang=None, zang=None)¶

Push a single node into the DB.

Parameters
inodint

Node number.

xfloat

X coordinate.

yfloat

X coordinate.

zfloat

Z coordinate.

xangfloat, optional

X angle.

yangfloat, optional

X angle.

zangfloat, optional

Z angle.

Examples

Update node 1 to have coordinates (10.0, 20.0, 30.0).

>>> nodes = mapdl.db.nodes
>>> nodes.push(1, 10, 20, 30)
>>> sel, coord = nodes.coord(1)
>>> coord
(10.0, 20.0, 30.0, 0.0, 0.0, 0.0)

ansys.mapdl.core.xpl.ansXpl¶

class ansys.mapdl.core.xpl.ansXpl(mapdl)¶

ANSYS database explorer class.

Examples

>>> from ansys.mapdl.core import launch_mapdl
>>> mapdl = launch_mapdl()
>>> xpl = mapdl.xpl
